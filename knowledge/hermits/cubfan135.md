# Cubfan135

## Overview

Cubfan135 (known as "Cub") joined Hermitcraft at the start of **Season 4** alongside GoodTimesWithScar, Iskall85, Rendog, and Welsknight — all recommended to the group from the Kingdomcraft server by PythonGB. He is a consistent, well-rounded contributor to the server across both technical and creative domains.
