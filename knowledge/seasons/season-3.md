# Hermitcraft Season 3

## Overview

Season 3 launched on **October 17, 2014** on Minecraft **1.8**, making it the first season to feature 1.8's new blocks, mobs, and mechanics. Notably, it used the **Amplified** world type — the only Hermitcraft season to do so — creating dramatically tall terrain. Several players who became long-term core Hermits joined this season, including EthosLab, Docm77, and ImpulseSV.

## Key Themes
